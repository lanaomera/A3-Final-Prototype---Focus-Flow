# A2-Final-Prototype---Focus-Flow

My Protoype is called Focus Flow, a musical composition coding prototype that generates personalized study music through EarSketch. 
Unlike a static Spotify playlist, my prototype uses your own mood, your current stress levels or overall vibe to create the perfect music for you. A simple user interface = mimimal distraction 

INSTRUCTIONS FOR USE

To run and experience your personalized soundscape:
1. Access the Code: Click the link provided under the A2 File location.
2. Execute: Select 'Run' to initiate the script.
3. Enter Emotion: When prompted, enter your desired mood.
  ⚠️ IMPORTANT: The user input is CASE SENSITIVE. You must type the emotion exactly as shown: Calm, Energised, Anxious, or Creative.
4. Enter Intensity: Select your stress/intensity level: 1 (Least) to 3 (Most).
5. Enjoy: Your personalized soundscape will begin playing instantly.
